추적 가능한 항목

NetWorthCast는 유동 자산과 비유동 자산 모두를 각각의 예상 수익률로 처리합니다. 이를 통해 총 순자산과 각 부분이 시간에 따라 어떻게 성장하는지에 대한 정확한 그림을 제공합니다.

유동 자산
현금, 고금리 저축, 401(k), IRA, 증권, 주식, 암호화폐

비교적 빠르게 현금으로 전환할 수 있는 자산. 경제적 자립에 포함되는 자산입니다.

비유동 자산
부동산, 차량, 수집품

실물 재산에 묶인 자산. 총 순자산에 기여하지만 즉시 지출에 사용할 수 없습니다.

각 자산에 고유한 예상 연간 수익률이 부여됩니다. 고금리 저축 계좌는 4.5%, 주식 포트폴리오는 8%, 암호화폐는 12%, 차량은 -15%일 수 있습니다. 이는 단일 혼합 수익률을 사용하는 것보다 훨씬 정확한 예측을 만듭니다.

NetWorthCast 추적 작동 방식

1. 자산 추가

이름, 현재 가치, 예상 수익률로 각 자산을 생성하세요. 예: "Fidelity 401k — $85,000 — 수익률 7%" 또는 "주거용 주택 — $420,000 — 상승률 3.5%."

2. 개별 수익률 설정

모든 자산이 고유한 비율로 복리 성장합니다. 현금은 주식과 다르게 수익을 내고, 차량은 감가상각됩니다. NetWorthCast는 현실적인 30년 예측을 위해 각각을 독립적으로 모델링합니다.

3. 주 계좌 지정

하나의 유동 계좌(일반적으로 당좌 또는 고금리 저축)를 주 계좌로 선택하세요. 이곳이 수입이 들어오고 지출이 나가는 곳입니다. 특정 달에 지출이 수입을 초과하면 이 계좌에서 먼저 부족분을 충당합니다.

4. 계좌 순서 조정

계좌를 원하는 순서로 드래그하세요. 순서가 캐스케이드 우선순위를 결정합니다 — 주 계좌가 소진되면 다음 유동 계좌에서 지출을 충당합니다.

단순 추적을 넘어서

오늘의 스냅샷만 보여주는 순자산 추적기로는 충분하지 않습니다. NetWorthCast는 재무 미래를 예측하여 지금 더 나은 결정을 내릴 수 있게 합니다.

  • 수입 및 지출 추적 — 반복 수입원과 월별 비용을 추가하여 현금 흐름을 확인
  • 일회성 이벤트 — 특정 미래 날짜에 주택 구매, 상속, 보너스 또는 자동차 할부 종료를 모델링
  • 자산 연동 수입 — 임대 수입을 부동산에, 배당 수입을 증권 계좌에 연결
  • 30년 예측 — 순자산이 매년 성장(또는 감소)하는 대화형 차트 확인
  • FIRE 계산 — 유동 자산이 경제적 자립 임계값을 넘는 시점을 자동으로 표시
  • 생활비 비교 — 이전이 궤적을 어떻게 변화시키는지 확인

프라이버시 우선 설계

재무 데이터는 민감합니다. NetWorthCast는 간단한 프라이버시 모델로 구축되었습니다: 데이터가 절대 기기를 떠나지 않습니다.

  • 모든 데이터는 브라우저의 localStorage에 저장됩니다
  • 서버, 데이터베이스, 클라우드 동기화 없음
  • 계정 생성이나 로그인 불필요
  • 재무 정보를 추적하는 분석 도구 없음
  • 첫 페이지 로드 후 완전 오프라인 작동

데이터를 완전히 소유합니다. 언제든 내보내거나 브라우저 저장소를 지워서 삭제할 수 있습니다.
